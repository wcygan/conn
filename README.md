A tool to test TCP connections.

## Usage

### Install

```bash
cargo install conn
```

### Run

```
Connect to an address over TCP

Usage: conn --ip <IP> --port <PORT>

Options:
  -i, --ip <IP>      The IP address
  -p, --port <PORT>  The port
  -h, --help         Print help
```
